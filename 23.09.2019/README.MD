# Päeva teemad
## Väljund
```
print "tekst mida soovid väljastada";
```
või
```
echo "tekst mida soovid väljastada";
```
## Muutujad
* Muutuja defineerimine
```
<?php
$muutujaNimi = väärtus;
?>
```
* Muutuja nimed on:
    * Loogilised
    * Tuleks kasutada täisnime
    * camelStyle-ina kirjutatud

* Muutuja kasutamine
```
<?php
echo "Mingi tekst ".$muutujaNimi."Ja veel mingi tekst";
?>
```
